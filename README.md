# Bird on TON BOT

- Auto play game
- Auto upgrade multiplier
- Auto view ads
- Support multi account
- Register BIRD on TON [here](https://t.me/BIRDTonBot/app?startapp=1375235586)

## Installation

``git clone https://github.com/himiko3939/bird.git
``

``
cd Bird-BOT
``

``
npm install
``

## Usage

### Edit data.txt 

``
nano data.txt
``

### Create screen

``
screen -S your_name_screen
``

### Run 

``
node main.js
``
